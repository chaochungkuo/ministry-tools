Apps for the Translation Work

Lasst das Wort Christi reichlich in euch wohnen in aller Weisheit... Kol. 3:16

The purpose of these applications is to assist the translation work in Lebensstrom. Please don't share these tools to non-translators who don't need them. Any feedback please send to Joseph Kuo

When a translator is working on the translation, it is always good to use the biblical terms and the sentences. Most of the expressions come from the Bible and we need to follow the Bible instead of developing a new expression. According to the input languages, there are two tools available:

Adding German scriptures to German documents

Adding German scriptures to English documents

Some notes as below:

  • The German verses are based on Elberfelder CSV 2006. About 1000 crucial verses are updated with Recovery Version Bible in German.
  • Please use only docx file. No doc files will be accepted.
